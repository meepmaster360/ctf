Installing
----------
Ideally, you should be able to just type:

    Choose directory
    
    git clone git@github.com:meepmaster360/ctf.git
    
    cd ctf


Using ctfmeep
----------
To run the aplication:

    $sudo bash ctfmeep.sh
    or
    $chmod +x ctfmeep.sh (only once)
    $sudo ./ctfmeep.sh
    #./ctfmeep.sh (root user)
