Installing
----------
Ideally, you should be able to just type:

    Choose directory
    
    git clone git@github.com:meepmaster360/ctf.git
    
    cd ctf


Using meepscan
----------
To run the aplication:
    $sudo bash meepscan.sh
    or
    $chmod +x meepscan.sh (only once)
    $sudo ./meepscan.sh
    #./meepscan.sh (root user)
